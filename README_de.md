---

## 📖 Projektübersicht

**GitTrend** ist eine statische Website, die automatisch neu erstellte Open-Source-Projekte auf GitHub sammelt, die in verschiedenen Zeiträumen (**Woche, Monat, Quartal, Jahr**) die meisten Sterne erhalten haben, ergänzt um einen speziellen Bereich für **Audio & Video KI**. Mithilfe der Google Gemini KI werden die englischen Originalbeschreibungen in **mehrere Sprachen** übersetzt und veredelt, um die „Hauptmerkmale & Highlights“ sowie „Praktische Anwendungsfälle“ jedes Projekts übersichtlich darzustellen.

Die Daten werden jeden Samstag über GitHub Actions aktualisiert, sodass Sie am Wochenende ausreichend Zeit haben, die neuesten Trends zu erkunden.

## ✨ Funktionen und Merkmale

| Funktion | Beschreibung |
| :--- | :--- |
| 🕐 **5-Stufen-Timeline** | Bietet Registerkarten: „Wöchentlich“, „Monatlich“, „Vierteljährlich“, „Jährlich“ und „Audio & Video KI“. Wechseln Sie mit einem Klick, um die Top 12 der angesagtesten Projekte zu sehen. |
| 🤖 **Gemini KI-Zusammenfassungen** | Nutzt Google Gemini 3.5 Flash, um Beschreibungen in flüssige Zielsprachen zu übersetzen und strukturiert in „Hauptmerkmale“ und „Anwendungsfälle“ aufzubereiten. |
| ☀️/🌙 **Sanfter Designwechsel** | Unterstützt Dunkel- und Hell-Designs mit weichen Übergangsanimationen und speichert Benutzereinstellungen automatisch im `localStorage`. |
| 🎨 **Glassmorphismus-Ästhetik** | Bietet ein wunderschönes Glassmorphismus-Design mit anpassbaren, leuchtenden Farbverlaufshintergründen für eine hochwertige Optik in beiden Modi. |
| 🏅 **Rangabzeichen** | Farbverlauf-Abzeichen in Gold, Silber und Bronze heben die Top-3-Projekte hervor, gefolgt von einer sauberen, gut ausgerichteten Rasteransicht. |
| 📱 **Responsive Design (RWD)** | Optimiert für Mobilgeräte, Tablets und Desktops. Die Tab-Navigation lässt sich auf mobilen Bildschirmen wischen, ohne das Layout zu verzerren. |
| ⚡ **Blitzschnelles Laden** | Basierend auf statischen JSON-Daten ohne API-Aufrufe zur Laufzeit, wodurch Ladezeiten im Millisekundenbereich erreicht werden. |
| 🔄 **Wöchentliche Updates** | Ein geplanter GitHub Actions Workflow läuft jeden Samstag um 08:00 Uhr UTC+8, um die neuesten Trend-Repositories abzurufen. |
| 💰 **100% Kostenlos** | Läuft vollständig im Rahmen der kostenlosen Kontingente der GitHub-API, Gemini-API, GitHub Actions und GitHub Pages. |

> 毎週土曜日に自動更新される GitHub トレンドプロジェクトタイムライン。Gemini AI を活用した詳細な要約により、オープンソース界の最新トレンドを素早く把握できます。

---

## 📖 プロジェクト概要

**GitTrend** は、GitHub 上で最近**「1週間」「1ヶ月」「3ヶ月」「1年」**に作成されたプロジェクト、および**「音声・映像 AI」**セクションで最もスターを獲得したトレンドオープンソースプロジェクトを自動的に収集する静的ウェブサイトです。Google Gemini AI を使用して、元の英語の解説を流暢な**多言語**に翻訳・要約し、各プロジェクトの「主な機能と特徴」および「実用的な活用シーン」を構造化して提示します。

ウェブサイトは毎週土曜日に GitHub Actions によって自動的に更新されるため、週末に最新のオープンソースのエネルギーをじっくりと探索することができます。

---

## ✨ 機能と特徴

| 機能 | 説明 |
| :--- | :--- |
| 🕐 **5段階の時間枠と専用セクション** | 「今週」「今月」「過去3ヶ月」「過去1年」「音声・映像 AI」の5つのタブを搭載。クリック1つで切り替え可能で、各期間のトップ12のトレンドプロジェクトを表示します。 |
| 🤖 **Gemini AI による多言語要約** | Google Gemini 3.5 Flash を使用し、英語の解説を自然で分かりやすい各言語に翻訳・要約。「主な機能」と「活用シーン」に整理します。 |
| ☀️/🌙 **スムーズなテーマ切り替え** | ダークモードとライトモードに対応。滑らかな遷移アニメーションを備え、`localStorage` でユーザーの好みを自動保存します。 |
| 🎨 **グラスモフィズムの美学** | Glassmorphism（グラスモフィズム）効果と適応型グラデーション発光背景を採用し、ダーク/ライト両モードでプレミアムな視覚体験を提供します。 |
| 🏅 **ランキングバッジ** | 上位3つのプロジェクトをゴールド、シルバー、ブロンズのグラデーションバッジで強調表示し、それ以降は整然としたグリッドで並べます。 |
| 📱 **レスポンシブデザイン (RWD)** | モバイル、タブレット、デスクトップに対応。モバイルデバイスでもタブナビゲーションが歪むことなくスムーズにスワイプできます。 |
| ⚡ **超高速読み込み** | 純粋な静的 JSON データ駆動により、実行時の API 呼び出しが不要で、ミリ秒単位で読み込みが完了します。 |
| 🔄 **毎週土曜日の自動更新** | GitHub Actions の定期タスクにより、毎週土曜日午前8時（日本時間）に最新データを自動取得します。 |
| 💰 **完全無料** | GitHub API、Gemini API フリーティア、GitHub Actions、GitHub Pages をすべて無料枠で運用しています。 |

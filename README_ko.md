---

## 📖 프로젝트 개요

**GitTrend**는 GitHub에서 최근 **1주일, 1개월, 3개월, 1년** 내에 생성되었거나 **오디오 및 비디오 AI** 분야에서 가장 많은 스타를 받은 인기 오픈소스 프로젝트를 자동으로 수집하는 정적 웹사이트입니다. Google Gemini AI를 활용하여 원래의 영어 설명을 자연스러운 **다국어**로 번역 및 요약하고, 각 프로젝트의 '핵심 기능 및 특징'과 '실제 적용 시나리오'를 구조화하여 제공합니다.

웹사이트는 매주 토요일 GitHub Actions를 통해 자동 업데이트되어, 주말 동안 최신 오픈소스 트렌드를 여유롭게 탐색할 수 있도록 돕습니다.

---

## ✨ 주요 기능 및 특징

| 기능 | 설명 |
| :--- | :--- |
| 🕐 **5단계 타임라인 및 섹션** | '최근 1주', '최근 1개월', '최근 3개월', '최근 1년', '오디오 & 비디오 AI'의 5개 탭을 지원합니다. 클릭 한 번으로 각 기간의 인기 프로젝트 Top 12를 쉽게 둘러볼 수 있습니다. |
| 🤖 **Gemini AI 다국어 요약** | Google Gemini 3.5 Flash를 사용하여 영어 설명을 자연스럽고 매끄러운 다국어로 번역하며, '핵심 기능'과 '실제 적용 시나리오'로 깔끔하게 정리해 줍니다. |
| ☀️/🌙 **부드러운 테마 전환** | 다크 모드와 라이트 모드를 모두 지원하며, 부드러운 그래디언트 전환 효과가 적용됩니다. `localStorage`를 통해 사용자의 설정이 자동으로 저장됩니다. |
| 🎨 **글래스모피즘 미학** | Glassmorphism(글래스모피즘) 디자인과 스스로 반응하는 그래디언트 배경 효과를 적용하여 다크/라이트 모드 모두에서 세련되고 프리미엄한 시각적 경험을 제공합니다. |
| 🏅 **랭킹 배지** | 상위 3개 프로젝트는 금, 은, 동 그래디언트 배지로 강조 처리되며, 나머지 프로젝트도 정돈된 격자 형태로 배열됩니다. |
| 📱 **반응형 웹 디자인 (RWD)** | 모바일, 태블릿, 데스크톱 화면에 완벽히 최적화되어 있습니다. 모바일 기기에서도 카테고리 탭이 잘림 없이 부드럽게 스와이프됩니다. |
| ⚡ **초고속 로딩** | 순수 정적 JSON 데이터를 기반으로 구동되어 런타임 API 호출이 전혀 없으며, 밀리초 단위의 빠른 페이지 로딩 속도를 보장합니다. |
| 🔄 **매주 토요일 자동 업데이트** | GitHub Actions 스케줄러를 통해 매주 토요일 오전 8시(한국 시간)에 최신 트렌드 데이터를 자동으로 수집합니다. |
| 💰 **100% 무료 운영** | GitHub API, Gemini API Free Tier, GitHub Actions, GitHub Pages의 무료 제공량 범위 내에서 완벽하게 구동됩니다. |

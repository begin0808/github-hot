## 📖 Project Overview

**GitTrend** is a static website that automatically aggregates newly created open-source projects on GitHub that have received the most stars over various periods (**Weekly, Monthly, Quarterly, Yearly**), along with a dedicated **Audio & Video AI** section. Using Google Gemini AI, it translates and refines project descriptions into natural **Traditional Chinese**, structurally presenting each project's "Core Features & Highlights" and "Practical Use Cases."

The website is updated automatically every Saturday via GitHub Actions, giving you ample time over the weekend to explore the latest trending projects.

## ✨ Features

| Feature | Description |
| :--- | :--- |
| 🕐 **5-Stage Timeline & Sections** | Includes five tabs: "Weekly", "Monthly", "Quarterly", "Yearly", and "Audio & Video AI". Switch with a single click to browse the top 12 trending projects for each period. |
| 🤖 **Gemini AI Refinement** | Uses Google Gemini 3.5 Flash to translate and refine descriptions into smooth Traditional Chinese, structured into "Core Features" and "Use Cases". |
| ☀️/🌙 **Smooth Theme Toggle** | Supports dark and light themes with smooth transition animations and remembers user preferences automatically using `localStorage`. |
| 🎨 **Glassmorphism Aesthetics** | Features a beautiful glassmorphism design and adaptive gradient glowing backgrounds, delivering a premium visual experience in both modes. |
| 🏅 **Ranking Badges** | Gold, silver, and bronze gradient badges highlight the top 3 projects, followed by a clean, well-aligned grid layout. |
| 📱 **Responsive Design (RWD)** | Fully optimized for mobile, tablet, and desktop screens. The tab navigation is swipeable on mobile devices without layout distortion. |
| ⚡ **Lightning Fast Loading** | Driven by static JSON data with no runtime API calls, achieving millisecond-level page load times. |
| 🔄 **Weekly Automated Updates** | Runs on a scheduled GitHub Actions workflow every Saturday at 08:00 AM UTC+8 to fetch the latest trending repositories. |
| 💰 **100% Free** | Fully runs on free tiers of the GitHub API, Gemini API, GitHub Actions, and GitHub Pages. |

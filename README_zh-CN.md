## 📖 项目简介

**GitTrend** 是一个静态网站，自动抓取 GitHub 上最近**一周、一个月、三个月、一年**内新建立，以及**影音与声音 AI 专区**中获得最多星星数的热门开源项目，并透过 Google Gemini AI 将英文描述翻译与精炼为**多国语言**，结构化呈现每个项目的「核心功能与特色」以及「实际应用场景」。

网站每周六由 GitHub Actions 自动更新数据，让你在周末有充裕的时间研究最新的热门开源能量。

## ✨ 功能特色

| 功能 | 说明 |
| :--- | :--- |
| 🕐 **五段式时间与专区** | 包含「最近一周」「最近一个月」「最近三个月」「最近一年」与「影音与声音 AI」五个标签页，一键切换浏览，每个时段收录前 12 名最热门的新开源项目 |
| 🤖 **Gemini AI 多国语言精炼** | 使用 Google Gemini 3.5 Flash 将英文描述翻译为自然流畅的各语系，包含「核心功能与特色」与「实际应用场景」 |
| ☀️/🌙 **双主题流畅切换** | 支持深色与亮色双主题，切换时具有柔和过渡动画，且会使用 `localStorage` 自动记忆用户偏好 |
| 🎨 **毛玻璃质感美学** | 采用 Glassmorphism 毛玻璃效果、自适应渐层发光动态背景，不论深色或亮色模式都具备极佳的视觉体验 |
| 🏅 **排名徽章** | 金、银、铜牌渐层标记前三名项目，后续项目整齐排版 |
| 📱 **响应式设计 (RWD)** | 完美支持手机、平板与桌机，分页导航按钮在移动设备上可流畅滑动且不压缩变形 |
| ⚡ **极速载入** | 纯静态 JSON 数据驱动，无需即时 API 呼叫，毫秒级载入 |
| 🔄 **每周六自动更新** | GitHub Actions 定时任务，北京时间每周六上午 08:00 自动抓取最新数据 |
| 💰 **完全免费** | GitHub API、Gemini API Free Tier、GitHub Actions、GitHub Pages 全部免费 |
